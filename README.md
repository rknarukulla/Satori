# Satori
Ask your Instagram data anything. Zero setup, zero code. Claude + Meta Graph API.

Satori is a lightweight, open-source Instagram analytics skill for Claude Cowork. Connect your Instagram Business account once, then ask questions in plain English — performance summaries, content patterns, caption drafts, content calendars. No Python, no local dependencies, no dashboards to navigate. Powered by Meta's official Graph API and Llama AI. Built by Zensplit, free for everyone.
